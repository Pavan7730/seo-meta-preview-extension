Privacy Policy

Last updated: February 2026

SEO Meta Preview is a browser extension designed to help users analyze on-page SEO elements such as meta titles, meta descriptions, headings, and search result previews.

Information Collection

This extension does not collect, store, transmit, or sell any personal or sensitive user data.

  • No personally identifiable information is collected
  • No browsing history is stored or tracked
  • No analytics or tracking scripts are used
  • No cookies are set

How the Extension Works

All data processed by SEO Meta Preview is accessed locally from the currently active web page in your browser. The extension reads publicly available HTML elements such as:

  • Meta title and meta description
  • Canonical and robots tags
  • Heading structure (H1–H4)
  • Open Graph image (if available)

This information is processed entirely within your browser and is never sent to any external server.

Permissions

The extension requires minimal browser permissions solely to function as intended:

  • activeTab – to read SEO-related data from the currently open webpage
  • scripting – to execute scripts on the active tab for analysis

Third-Party Services

SEO Meta Preview does not integrate with or send data to any third-party services, APIs, or analytics platforms.

Data Security

Since no user data is collected or transmitted, there is no risk of data leakage or misuse.

Changes to This Policy

This privacy policy may be updated from time to time. Any changes will be reflected on this page with an updated revision date.

Contact

If you have any questions or concerns about this privacy policy, you may contact the developer via the Chrome Web Store listing.
